Code As Art

Spring 2020

Interactive Art

This is an interactive art piece where the user follows a star around the 
canvas in order to reveal pieces of a poem while witnessing the phases of 
the moon. The evolution of the moon's shape is in dialogue with the poem 
about our own internal shifts in alignment with lunar energy. The written 
portion could also be interpreted as a guided meditation or directed 
visualization. 

Code:

function setup() {
  createCanvas(900, 700);
  textAlign(CENTER);
  textSize(20);
} // end setup


function draw() {
  background(0);
  
  levelOne();
  
  text(("Phasing: " + score), width/2, 40);
  fill(255);
  

} // end draw
